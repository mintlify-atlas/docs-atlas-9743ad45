# Introduction to Findex

Findex is a highly customizable application launcher written in Rust and uses GTK3. Designed for Linux power users, Findex combines blazing-fast fuzzy search with deep customization options to help you launch applications exactly the way you want.

## What is Findex?

Findex is a lightweight, keyboard-driven application launcher that sits in the background and springs to action when you need it. Unlike traditional application menus, Findex uses intelligent fuzzy search to help you find and launch applications with just a few keystrokes.

## Key features

- **Customization using CSS**: Apply custom styles to any UI element using standard CSS properties
- **Plugin system**: Write plugins in Rust to add new functionality and search sources
- **Custom shortcuts**: Define keyboard shortcuts for triggering plugins and executing results
- **Fast fuzzy search**: Powered by Sublime-style fuzzy matching for accurate results
- **Configurable behavior**: Control window size, result count, scoring thresholds, and more via `settings.toml`
- **Wayland and X11 support**: Works on both display servers with appropriate configuration

## Architecture

Findex runs as a daemon (`findex-daemon`) that stays in the background, waiting for your hotkey. When triggered, it launches the main Findex window for searching and launching applications. This architecture keeps memory usage low and ensures instant response times.

## System requirements

- **Operating System**: Linux (X11 or Wayland)
- **Dependencies**: GTK3, libkeybinder3
- **Build Requirements**: Rust v1.66.0 or later (if building from source)
